Common questions

Questions before you download?

These quick answers cover setup, offline usage, and team access so you can decide faster.

Can we use JamsGadget while offline? +

No. JamsGadget requires an internet connection to access and update inventory data in real time.

How long does setup usually take? +

Most small shops can upload inventory and start logging sales in under 30 minutes.

Can multiple staff members use JamsGadget? +

Yes. You can create multiple user accounts so different staff members can access the same store data.

Can we track installment payments? +

Yes. You can record installment transactions, track balances, and monitor payment history per customer.

Do you currently support iOS and desktop? +

For now, JamsGadget is available on Android only. iOS and desktop support are planned for a future release.
